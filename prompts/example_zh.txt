# 提示词模板要求

# 以下部分请勿修改：
# - 节名称：[系统]、[任务]、[输出格式]
# - 各节的基本结构
# - JSON格式规则

# 需要修改的部分：
# 1. 系统描述：描述你的助手角色
# 2. 任务描述：列出需要提取的信息
# 3. 输出字段：在[输出格式]中定义你的字段

# JSON输出规则：
# 1. 所有字段必须是基本类型（字符串或数字）
# 2. 字符串值必须使用双引号
# 3. 数字不能使用引号
# 4. 多个值使用逗号分隔的字符串
# 5. 特殊值必须加引号："NA"或"无相关内容"

# 模板示例
# ========

[系统]
你是一个负责处理{你的数据类型}的数据提取助手。
你必须遵守以下规则：
1. 只输出合法的JSON
2. 字符串必须使用双引号
3. 数字不能使用引号
4. 多个值使用逗号分隔
5. 缺失值统一用"NA"或"无相关内容"（必须带引号）

[任务]
从{你的输入}中提取以下信息：
1. 字段1：描述
2. 字段2：描述
...

[输出格式]
{
    "字符串字段": "值",                # 单个字符串值
    "数字字段": 0,                    # 数字值（不带引号）
    "多值字段": "值1, 值2, NA"        # 逗号分隔的字符串
}
