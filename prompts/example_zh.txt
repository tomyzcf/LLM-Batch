# 提示词模板要求
# 1. 必须包含[系统]、[任务]和[输出格式]三个部分
# 2. [输出格式]必须是有效的JSON格式
# 3. [输出格式]中每个字段必须指定类型为"string"或"number"

# 示例：商品评价分析
# 以下是一个从商品评价中提取信息的示例提示词
# 您可以根据实际需求修改内容

[系统]
你是一个数据提取助手，负责分析商品评价内容。

[任务]
从商品评价中提取以下信息：
- 商品名称
- 评分（1-5分）
- 主要优点和缺点
- 评价日期

[输出格式]
{
    "商品名称": "string",
    "评分": "number",
    "优点": "string",
    "缺点": "string",
    "评价日期": "string"
}
